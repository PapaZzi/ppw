Marvelous Design.

Marvelous Design is helpful in making 3D character models. I can use this software to design many interesting and complex clothes and dress the characters.

Marvelous Design can be used to create realistic clothes through a variety of built-in cloth materials. I can easily refine and modify on other 3D software to achieve better results. It enables me to save time and increase efficiency.

Zbrush.

ZBrush is a kind of 3D engraving software. I can use ZBrush to carve a lot of details and shapes on 3D models. In addition, I can use ZBrush to color the model and then use RizomUV to make a material map.

ZBrush can also transform high model into low model with reasonable wiring. If I need to make subsequent animation or insert it into other 3D software. ZBrush is important to me.

×
×

RizomUV.

Due to the model made by using ZBrush doesn't have UV. If I want to make a material map for the model, I can use RizomUV to create the UV for the model.

Rizomuv has powerful functions. I can choose to automatically expand into one or several pieces or dozens of pieces UV. At the same time, even if it is a general laptop, RizomUV can also handle about 100000 face models. This allows me to establish my 3D model faster.

3DMax.

I can skillfully use 3DMAX to make regular geometric models. 3DMAX is much better and more convenient than most of 3D software when making geometry.

But in addition, I use 3DMAX more for rendering. 3DMAX supports many excellent rendering plug-ins, such as vary2, Cr and so on. Through 3DMAX, I can render many exquisite pictures.
