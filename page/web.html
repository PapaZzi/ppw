CSS Glow Animation.

Many times, static web pages will make users feel very boring. To change this, I tried to add some CSS animation effects. This CSS glow animation is one of them.

I applied this CSS glow animation effect to the title. This enables the title to display a glowing effect, thus attracting the user's attention at the first time. I think it's very helpful to attract users.

CSS Moving Animation.

Most of the time, I'm thinking about how to do the Welcome page better. It can't be too complicated, but it also needs to be interesting enough. With continuous learning, I gradually learned this animation effect.

On a black background, there will be large or small white stars flashing at the bottom and moving upward. At the same time, the title and buttons are in circular motion. I think it is a simple but effective approach.

CSS Transiton Animation.

Many times, I need to make gallery pages to show some pictures and materials. It would be boring to put it directly on the page horizontally and vertically.

I use CSS transition animation to place the pictures irregularly, like photos scattered on the ground. When the user moves the mouse over the picture, the picture will be enlarged and straightened. I think it's much better than putting it directly.

Back To The Top Function.

When I make my own web pages, I find it troublesome to pull the page to the bottom and then back to the top. I even feel tired after doing it a few more times. I think users may have the same feelings.

In order to bring users better experience, I used JavaScript to make a back-to-the-top function. As long as the user clicks the button in the lower right corner of the page, he can return to the top all at once. I think this will make users feel very convenient and free.

360Degree Img.

This is the technology I used when making toy photograph website. I want to show a toy more comprehensively, and I don't want to use too many pictures to occupy too much of the page. When the user moves left and right on the picture with the mouse, the toy will also move left and right with it.

This technology is mainly based on JavaScript and I need to take a large number of photos at the same time so as to form the frame by frame effect. Although creating one 360degree img will take a lot of time, I think it's worthy to do so.

CSS Animation.

This is an attempt at CSS animation. I mainly want to put one of my ideas into practice.

I think it's too simple to put a picture as a business card or self introduction, so I use this CSS animation to make it more interesting. When the user moves the mouse to the specified position, the animation will begin to play. Through this animation, users can have a basic understanding and get contact information.
