Adequate Guidance.

When I'm using a brand new app myself, I often wonder what to do next, like what's the use of this button. To avoid this situation, I add adequate guidance on the application to enable users to use it easily and smoothly.

When the user moves the mouse to the corresponding icon or button, the detailed text will appear to tell the user the specific functions.

Interesting Interaction.

In fact, in the process of using other applications or web pages. I find that interesting interaction design can attract more attention of users. So that the user is not bored. therefore, I designed some interactive animations in my application.

The interaction design I make mainly focuses on the interaction with the mouse. When the user moves the mouse over the icon, there will be corresponding changes. I hope that users can like my application better because of these interaction designs.
